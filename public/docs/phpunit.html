PHPUnit

1. Install dependencies

  • Install phpunit via composer

composer require --dev phpunit/phpunit

Note: We are installing phpunit package as dev dependency because we don't need it for proper functioning of app

2. Tweak project structure

  • In project root create dir names tests/

  • Now, also in project root, create a file named phpunit.xml

  • Previously created file should look like this

<?xml version="1.0" encoding="utf-8" ?> <phpunit color="true" bootstrap="vendor/autoload"> <testsuites> <testsuite name="unit"> <directory>tests/unit</directory> </testsuite> </testsuites> </phpunit>

Note: An XML file is nothing more but configuration file that tells unit interpreter where to look for tests

  • As the last step, in dir tests/ create sub-dir unit/

3. Setup test structure

  • Let's assume that we have class named User in our project

  • First, class responsible for testing methods inside User class would be named UserTest

Note: A general pattern of naming test classes is BaseNameOfClass + Test, that's it, that would be naming convention for test classes

  • Our User class looks like this

<?php declare(strict_types=1); namespace User; class User { public function __construct() {} public function greetUser(string $username): string { return "Hello " . $username . ", how are you?"; } }

  • In tests/unit dir we created class UserTest that looks like this

<?php declare(strict_types=1); namespace Tests/Unit; use PHPUnit\Framework\TestCase class UserTest extends TestCase {}

4. Write test

  • For the first step, we need to make test method inside UserTest

  • Method can be named it_greets_a_user() and it should return void

class UserTest extends TestCase { it_greets_a_user(): void {} }

  • Now we can create a new object of User class

it_greets_a_user(): void { $user = new User(); }

  • After creating a user object, we can create expected output of greetUser() method

  • Hello John, how are you?

  • Now we can call method and make equal assertion

it_greets_a_user(): void { $user = new User(); $expectedOutput = "Hello John, how are you?"; $this->assertEquals($expectedOutput, $user->greetUser("John")); }

  • Now, our class looks like this

<?php declare(strict_types=1); namespace Tests/Unit; use PHPUnit\Framework\TestCase class UserTest extends TestCase { test_it_greets_a_user(): void { $user = new User(); $expectedOutput = "Hello John, how are you?"; $this->assertEquals( $expectedOutput, $user->greetUser('John') ); } }

  • Finally, if we run our test with ./vendor/bin/phpunit UserTest we should get test results back

PHPUnit 11.3.3 by Sebastian Bergmann and contributors. Runtime: PHP 8.2.12 . 1 / 1 (100%) Time: 00:00, Memory: 22.86 MB OK (1 test, 1 assertion)

  • That's it, this was a successful unit test
