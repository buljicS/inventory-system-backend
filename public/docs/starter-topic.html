Upgrading

If you want to try out this project, I suggest installing all dependencies from composer.lock file and then follow General setup documentation to set up and have all up and running

composer install

However, if you wish to set up a project and upgrade everything, I suggest that you make a copy of the project and go step by step, upgrading a single dependency at the time.

  • For PHP check PHP Watch

  • For possible Slim upgrade, check SlimPHP Docs (Upgrade Guide)

  • For all other dependencies, check packagist or official git repo
